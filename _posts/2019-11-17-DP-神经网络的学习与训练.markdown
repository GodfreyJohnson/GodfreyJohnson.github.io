# 从数据中学习

- 难点：如何让机器和人一样自动识别5这个手写的数字？
- 方案：
    - 机器学习：先撮特征量（SIFT/SURF/HOG），再将转换后的向量，通过SVM/KNN等分类器进行机器学习。 
    - 深度学习：直接从原始数据中获得目标结果输出。优点是，所有的问题可以用相同的流程进行解决。
